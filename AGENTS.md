# Project Agent Instructions

- 在本项目中处理任何任务前，优先读取 `_kb_sync/KB_CACHE.md`（若存在且非空）作为知识库上下文。
- 若 `_kb_sync/KB_CACHE.md` 不存在或为空，再读取 `_kb_sync/KB_CONTEXT.md`。
- 若上述两者都不存在或为空，再回退到常规项目文件检索。
- 本项目中生成、改动代码时，需要同步生成备注。备注使用中文。
- test 路径下，是所有测试场景。正式场景不在本目录下。
- sciprt 路径下，是所有功能脚本。其中 `sciprt\config` 下是配置文件
- assets 路径下，是美术资源。
- 对美术资源的配置，应当基于固定文件路径，配置对应资源的文件名称，而不是配置完整资源路径。
